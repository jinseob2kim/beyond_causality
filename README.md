# beyond_causality

## 인과추론 vs 장(Field) 중심 역학 — 존재론적 가정 비교

이 섹션은 인과추론과 장(Field) 중심 프레임이
**무엇을 ‘존재하는 것’으로 가정하는지**를 명시적으로 대비한다.
이는 방법론의 차이가 아니라, **세계에 대한 기본 전제의 차이**다.

---

### 1. 개입(Treatment)의 존재론

| 항목 | 인과추론 | 장(Field) 중심 |
|-----|---------|---------------|
| 개입의 정체성 | 잘 정의되고 반복 가능한 동일 객체 | 맥락·상태에 따라 달라지는 연산 |
| 수학적 역할 | 외생적 변수 $$A$$ | 상태 이동 연산자 |
| 암묵 가정 | $$A = A$$ | $$A(s_1) \neq A(s_2)$$ 가능 |
| 문제 발생 시 | 측정오차/혼란 | 구조적 이질성 |

---

### 2. 반사실(Counterfactual)의 지위

| 항목 | 인과추론 | 장(Field) 중심 |
|-----|---------|---------------|
| 핵심 객체 | \(Y(1), Y(0)\) | 관측 가능한 상태 전이 |
| 반사실의 지위 | 항상 개념적으로 존재 | 존재하지 않을 수 있음 |
| 비관측 vs 비존재 | 비관측으로 취급 | **비존재(non-existence)** 인정 |
| 의미 | 추정 대상 | 정의 불가능하면 질문 무효 |

---

### 3. 시간과 동역학

| 항목 | 인과추론 | 장(Field) 중심 |
|-----|---------|---------------|
| 시간 처리 | 단계적, 모델로 보정 | 기본 구성 요소 |
| 상태-개입 관계 | 순차적이지만 분리 가능 | 상호 재구성 |
| 전제 | 반사실 경로 정의 가능 | 경로 자체가 분석 대상 |
| 대표 방법 | MSM, g-formula | 전이 커널, 궤적 |

---

### 4. Positivity / Overlap의 의미

| 항목 | 인과추론 | 장(Field) 중심 |
|-----|---------|---------------|
| 가정의 성격 | 필수 조건 | 관측 대상 |
| 위반 시 해석 | 분석 실패 | 구조적 경계 |
| 대응 | trimming, restriction | 경계/절벽/금지구역 기술 |
| 의미 | 비교 불가 | 시스템 제약 노출 |

---

### 5. 관측자(분석가)의 위치

| 항목 | 인과추론 | 장(Field) 중심 |
|-----|---------|---------------|
| 관측자 지위 | 시스템 외부 | 시스템 내부 |
| 분석의 영향 | 데이터 생성과 독립 | 데이터 생성에 피드백 |
| 위험 점수/모델 | 설명 도구 | 장을 변형하는 연산 |
| 결과 | 고정된 효과 | 적응적 구조 |

---

### 6. 추정 대상(Estimand)

| 항목 | 인과추론 | 장(Field) 중심 |
|-----|---------|---------------|
| 주 대상 | 평균 효과 (ATE/ATT) | 장의 기울기·곡률 |
| 단위 | 스칼라 | 구조/지도 |
| 질문 | “얼마나 효과가 있나?” | “어디서 어떻게 움직이나?” |
| 손실 | 구조 정보 | 평균 요약 없음 |

---

### 7. 실패의 해석

| 항목 | 인과추론 | 장(Field) 중심 |
|-----|---------|---------------|
| 가정 붕괴 | 편향/불안정 | 의미 있는 관측 |
| 데이터 문제 | 제거 대상 | 분석 대상 |
| 결과 불안정 | 결론 약화 | 구조 민감성 신호 |

---

### 요약

- 인과추론은 **고정된 개입 + 외부 관측자 + 반사실 세계**를 전제한다  
- 장(Field) 프레임은 **동역학적 시스템 + 내부 관측자 + 구조적 제약**을 전제한다  
- Positivity 위반은 방법론 실패가 아니라  
  **이 두 세계관의 불일치가 드러난 첫 징후**일 수 있다
